# VisorEssentialsPlus Neoforge 1.21.1 Unoffical

This is the unofficial NeoForge 1.21.1 version of VisorEssentialsPlus.

> [!WARNING]
> This addon will **NOT** work with any official release of Visor. It only works with `Visor-0.4.0-NeoForge-1.21.1-Unofficial`.

In this exapanded version it adds custom buttons, a creative inventory menu only viewable when in creative mode, and support for
other addons that rely on VisorEssentialsPlus to function.

## Compatibility

- Minecraft 1.21.1
- NeoForge 21.1.234 or newer
- `Visor-0.4.0-NeoForge-1.21.1-Unofficial` only
- Java 21

Based on the original VisorEssentials project by PhoenixRa.
